# ITRGenie

Tax filing SaaS application - File your ITR online with ease.

## Features
- Simple ITR filing
- Tax calculation
- Document upload
- Expert support

## Tech Stack
- Next.js
- Tailwind CSS
- Vercel (Hosting)

## Setup
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
